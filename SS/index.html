1

Uvod u JS

Opis: web stranica pomocu skripte(JS) mijenja boju pozadine ovisno o dobi dana; pokazuje trenutno vrijeme i govori koje je dobar dana; dodatno: pretvorba HRK u euro.

2

Operatori u JS

Opis: web stranica pomocu skripte(JS) izvrasava racunske operacije; ispisuje parne i neparne brojeve (i u konzolu); pretvaranje u hex., binarni sustav; izracun po zelji.

3

Ugradene metode

Opis: web stranica pomocu skripte(JS) otvara novi prozor(window.open) kojem su pozadina, tekst, boja teksta i velicina teksta parametri funkcije; racunanje potencija.

4

Polje, objekti

Opis: upis u polje, ispis polja; spajanje polja; koristenje objekata MATH i WINDOW; vracanje na prethodnu stranicu; osvjezavanje stranice; provjera kolacica.

5

Kredit

Opis: pomocu forme prati se upis podataka korisnika o kreditu, web stranica racuna mjesecnu ratu, ukupni troska kredita i ima ispis otplate.

6

JS dogadaji

Opis: stranica o procesorima, memorijama i diskovima; stranica koristi JS dogadaje da za promjenu boje pozadine kod hovera, kod klika ...

7

Kolacici

Opis: vjezba 6, ali slike se mogu zoomirati; stranica memorije zasticena je lozinkom, uredivanje teksta o diskovima, kolacici(cookies) pamte odredene podatke.

8

JSON

Opis: pomocu JS jezika se pretvara JSON string u objekt i ispisuje se na zaslon; isto tako, obratno, upisani objekt u input se pretvara u JSON string i ispisuje.

9

AJAX

Opis: pomocu AJAX objekta, 1. ispisuje se trenutno vrijeme, 2. upisana imena se pretvaraju u mala slova, 3. ispis osobnih podataka.
